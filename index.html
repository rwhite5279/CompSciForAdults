Welcome.

This website is designed to provide a five lesson Introduction to Computer Science and Computer Programming course for adults with little or no previous experience in computer science. Lessons are typically about two hours in length. The materials in this course are designed to be used by an experienced instructor as a guide in conducting the course.

Each lesson is designed to led by a competent instructor, and should take approximately two hours to complete. Lessons include non-computer activities, discussion prompts, links to videos, and a short introduction to programming using the Python language.

Resources

The class will make use of a number of resources:

  • Your computer running Windows, macOS, or Linux
  • A text editor (you'll install this as part of the course)
  • Python3 (you'll install this as part of the course)
  • YouTube videos
  • Online readings
