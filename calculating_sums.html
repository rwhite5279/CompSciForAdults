Calculating Sums

One way to calculate the sum of a series of numbers is to set up a counting loop that goes through each number in the series. Every time you identify a new number, add it to the sum of all the previous numbers that you've seen.

In English, we might write the algorithm like this:

  1. Go through every number from 1 to 100
  2. For each number, add it to the sum of all the numbers before
  3. Once you've gone through all the numbers, the answer is that sum

In Python, the algorithm could be coded like this:

sum = 0 for number in range(1,101): sum = sum + number print(sum)

A quicker way to identify that sum of numbers is to use a formula. If you look at the list of numbers, you can see that the numbers at the beginning and end of the list can make pairs:

1, 100
2, 99
3, 98
4, 97
.
.
.

Each of those pairs of values adds up to be 101, and there are 50 pairs. So, if n = the number of items in our list, (50) * (n + 1) gets us our answer. 50 is just half of n, so the final formula, for any range of values 1..n, is

Sum of value from 1 to n = (n / 2) * (n + 1).

One of the challenges of coming up with an algorithm is determining whether it works for a variety of values. Does this algorithm (formula) also work for odd numbers of values, say 1-99? Try it out and see! You can test the formula by trying it with some different values, or you can prove it by doing a mathematical analysis.
