  5. Intro to Hypertext Markup Language (HTML)
    1. Markup language: <     >
    2. Tags for pages:
      • <html>     </html>
      • <head>     </head>
      • <body>     </body>
    3. Tags for text:
      • Paragraphs: <p>     </p>
      • Emphasized (italicized): <em>     </em>
      • Strong (bold): <strong>     </strong>
      • Line break: <br />
      • Headers: <h1>     </h1>, <h2>     </h2>, etc.
    4. Links: <a href="target URL">link text</a>
    5. Images: <img src="image_filename" />
    6. Logical Divisions: <div>     </div>
  6. Write a webpage

    Write your first webpage

    Download this template1.html file and modify it as you wish. Make sure that your page include fours divs—header, menu, contentwrapper, and footer—and populate those divs with information as you wish.

      Modify your webpage

      Download this template2.html file and identify the lines that have been modified. Adjust your template1.html file to match it where needed. You'll also need to download showHide.js and styles.css and place those files in the same directory as your html files for them to work.
